## a - z

```
a（全选所有文件）
b（解压文件，默认解压到对侧目录，删除路径后变为解压到当前目录）
c（创建目录）
d（打开书签，使用菜单首字母或带下划线字母定位）
dc（配置书签）
dr（从书签删除当前目录）
e（或f4，编辑光标所在文件，只能配一个编辑器，一般用来编辑文本文件）
f（提示复制、移动相关的更多操作）
g（提示标签相关的更多操作）
gg（到第一个文件）
数字+gg（跳到第n个文件）
g+数字（定位到第n个标签页）
h（上一层目录）
i（创建文件，输入菜单首字母定位）
j（向下）
k（向上）
l（同回车，进入目录、打开文件、将压缩文件当目录打开）
m（添加标签）
n（选择最近访问过的目录）
o（调用explorer的右键，输入菜单中带下划线的字母定位）
p（压缩文件到对侧目录）
q（在对侧窗口预览光标所在文件）
r（或f2*，重命名所有选定文件）
s（切换文件排序方法）
t（打开新标签页）
u（或gc，关闭当前标签页）
v（切换列视图）
w（或f3，在新窗口预览选定文件）
x（或del，删除所有选定文件到回收站）
y（复制选定文件的文件名）
z（提示更多窗口相关操作）
zv（切换水平切分、垂直切分）
zz（切换半屏、全屏）
```

## A - Z

```
A（同Shift+f3，在新窗口预览光标所在文件）
B（将所有子目录的文件全部显示在当前目录）
C（使用内置工具比较2个选定文件内容）
D（到桌面目录）
E（进入cmd）
F（在单独进程打开搜索对话框）
G（到最后一个文件）
H（后退）
I（或alt+f3，使用外部查看器打开文件）
J（选择/反选 光标所在文件，然后光标移动到下一个文件）
K（选择/反选 光标所在文件，然后光标移动到上一个文件）
L（前进）
M（打开内置的“比较两侧目录内容”窗口）
N（或gt，定位到后一个标签页）
O（打开按扩展名过滤文件对话框）
P（或gT，定位到前一个标签页）
Q（启用、禁用菜单栏）
R（打开批量重命名对话框对所有选定文件重命名）
S（无）
T（打开驱动器列表）
U（或ctrl+\，到所在磁盘的根目录）
V（界面调整）
W（使用外置工具比较选定的2个文件/目录内容）
X（或shift+del，彻底删除所有选定文件）
Y（复制选定文件的文件路径）
Z（打开配置窗口）
```

## ctrl+a - ctrl+z

```
ctrl+a（全选所有文件）
ctrl+b（或PgUp，上一页）
ctrl+c（复制所有选定文件，如果没有选定文件，则复制光标所在文件，下同）
ctrl+d（同shift+\，取消选择所有文件）
ctrl+e（使用外置工具解压文件）
ctrl+f（或PgDn，下一页）
ctrl+g（打开更改属性窗口）
ctrl+h（在缩略图模式中向左移动光标）
ctrl+i（同tab，切换焦点到对侧窗口）
ctrl+j（同j，在缩略图模式中向下移动光标）
ctrl+k（同k，在缩略图模式中向上移动光标）
ctrl+l（在缩略图模式中向右移动光标）
ctrl+m（比较两侧目录内容，并选定不同的文件）
ctrl+n（连接到ftp）
ctrl+o（无）
ctrl+p（无）
ctrl+q（打开内部命令帮助窗口）
ctrl+r（刷新）
ctrl+s（显示当前目录下所有子目录的大小）
ctrl+t（无）
ctrl+u（对调两个窗口内容）
ctrl+v（粘贴已经复制的文件）
ctrl+w（使用自定义外部工具比较选定文件或目录）
ctrl+x（剪切所有选定文件）
ctrl+y（复制选定文件的所有列）
ctrl+z（编辑备注）
```
